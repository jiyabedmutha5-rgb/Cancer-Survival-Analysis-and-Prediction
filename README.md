# Cancer Survival Analysis and Prediction

## Project Overview

This project focuses on analyzing cancer patient data to identify factors affecting survival outcomes and to build predictive models that can assist in understanding patient prognosis.

The objective is to combine statistical analysis and machine learning techniques to explore patterns in cancer survival data and generate meaningful insights.

## Business Problem

Healthcare organizations collect large volumes of patient information, but extracting actionable insights from this data remains challenging.

This project aims to:

* Analyze survival-related factors
* Identify important patient characteristics
* Explore survival trends
* Develop predictive models for survival outcomes

## Key Analysis Performed

* Exploratory Data Analysis (EDA)

* Visualization of Key Trends

## Results

The project demonstrates how statistical and machine learning techniques can be used to analyze cancer survival data and support data-driven decision making.
